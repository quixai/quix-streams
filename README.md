# Quix streaming SDK module

## Overview

This module is the c++ implementation of the next gen Quix SDK library for [Quix.ai](https://quix.ai/).

## Linux development

### How to build?

```sh
    mkdir build && cd build && cmake .. && make -j4
```

### How to test?

```sh
     ./build/test/QuixStreaming_tst  
```
